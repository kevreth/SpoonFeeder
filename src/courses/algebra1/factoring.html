Number of Terms Factoring Rule Name Formula
>1 Greatest Common Factor (GCF) If \(d\) is the GCF of \(a\) and \(b\), then \(a + b = d(\frac{a}{d} + \frac{b}{d})\)
2 Difference of Squares \(a^2 - b^2 = (a + b)(a - b)\)
Sum of Cubes \(a^3 + b^3 = (a + b)(a^2 - ab + b^2)\)
Difference of Cubes \(a^3 - b^3 = (a - b)(a^2 + ab + b^2)\)
3 Square of a Binomial Sum \(a^2 + 2ab + b^2 = (a + b)^2\)
Square of a Binomial Difference \(a^2 - 2ab + b^2 = (a - b)^2\)
Reverse FOIL method \(x^2 + bx + c = (x + m)(x + n)\)
where
\(m + n = b\), and
\(mn = c\)
AC method \(ax^2 + bx + c = (mx + n)(px + q)\)
where
\(mp = a\),
\(mq + np = b\), and
\(nq = c\)
4 Grouping \(ac + ad + bc + bd \\ = a(c + d) + b(c + d) \\ = (a + b)(c + d)\)
