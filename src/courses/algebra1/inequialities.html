Property Name Expression
Reflexive Property \( a \leq a \) and \( a \geq a \)
Transitive Property If \( a \leq b \) and \( b \leq c \), then \( a \leq c \)
Addition Property If \( a \leq b \), then \( a + c \leq b + c \) for any number \( c \)
Subtraction Property If \( a \leq b \), then \( a - c \leq b - c \) for any number \( c \)
Multiplication Property (Positive Number) If \( a \leq b \) and \( c > 0 \), then \( ac \leq bc \)
Multiplication Property (Negative Number) If \( a \leq b \) and \( c < 0 \), then \( ac \geq bc \)
Division Property (Positive Number) If \( a \leq b \) and \( c > 0 \), then \( \frac{a}{c} \leq \frac{b}{c} \)
Division Property (Negative Number) If \( a \leq b \) and \( c < 0 \), then \( \frac{a}{c} \geq \frac{b}{c} \)
